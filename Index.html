Admissions

Simple steps to apply

How to Apply

  1. Register using the quick form above or the full application form in Admissions page.
  2. Provide basic documents (ID, previous study if applicable).
  3. Attend an orientation & placement assessment (online).
  4. Receive acceptance letter and choose class schedule.

Eligibility

Students of all ages are welcome. Some advanced tracks require prior knowledge; check each course page for details.
